
# Web Photobooth Siap Deploy

## Deploy
1. Clone repo atau download ZIP
2. Drag & drop folder `photobooth/` ke [Netlify](https://app.netlify.com/drop)
3. Atau upload ke GitHub dan aktifkan Pages (`Settings > Pages`)
4. Buka URL untuk langsung mencoba photobooth

## Catatan
- Semua frame & sticker bawaan ada di folder `assets/frames` & `assets/stickers`
- Bisa upload frame/sticker sendiri langsung dari UI
